# Sinhala SEO Slugifier

A high-performance utility to convert Sinhala Unicode text into SEO-friendly URL slugs.

## Installation
```bash
npm install sinhala-slugify
```

## Usage
```JavaScript
const { sinhalaSlugify } = require('./src/index');
const slug = sinhalaSlugify("අලුත් පුවත් 2026");
console.log(slug);
```
